AI Resume Analyzer:
An AI-powered resume analyzer that matches your 
resume with job descriptions and gives ATS score.

Link:-   https://ai-resume-analyzer-bxddf8rlhrfyej88zfhvff.streamlit.app/

 Features
- ATS Match Score
- Skills Gap Analysis
- Resume Improvement Suggestions
- AI Interview Questions
- Interview Probability Prediction

 Tech Stack
- Python
- Streamlit
- Scikit-learn (TF-IDF + Cosine Similarity)
- Groq LLM API
- Plotly

 Installation
pip install -r requirements.txt
streamlit run app.py
